    Build Signature

    Also can see the example on the right-side to see how to make it on Javascript

    How to Generate Signature on Postman Pre-Script

    secretKey =
  "sk-760a8a0c8826b2e9dd8fe3ed0c66976f0c3d4a0443056674b7c132a25ddbbaf6eed1a410e02609a0a9791d7683c5de83860574bf6a92ddf8ea5a14360a96ac3c";
// Get the current timestamp in RFC3339 format
var timestamp = new Date().toISOString();
// Get the request method and URI
var method = pm.request.method;
var uri = pm.request.url.getPathWithQuery();
// Concatenate the method, URI, and timestamp
var signaturePayload = method + "\n" + uri + "\n" + timestamp;
// Create a HMAC SHA256 hash of the signature payload
var hash = CryptoJS.HmacSHA256(signaturePayload, secretKey);
// Encode the hash in base64
var signature = CryptoJS.enc.Base64.stringify(hash);
// Set the timestamp and signature as environment variables
pm.environment.set("request_timestamp", timestamp);
pm.environment.set("request_signature", signature);

// use the environment on your request.

    Format

    Format Example
    HTTP METHOD + "\n" + URI + "\n" + TIMESTAMP POST
    /v1/payins
    2024-05-10T16:42:22.802Z

    Sign the Signature

    signaturePayload= "POST\n/v1/payins\n2024-05-10T16:42:22.802Z"
    var hash = CryptoJS.HmacSHA256(signaturePayload, <Secret Key>)
    signature = CryptoJS.enc.Base64.stringify(hash);

    Now finally, add the signature to every request header when calling the API for Zenith

    Xenith-Request-Signature = signature

    Query Parameters

    Parameter Required Type Description
    limit N string The limit of the result (default 50)
    cursor N string The offset of the result (default 0)
    statuses N string Filter result by list of statuses, enum of ["ACTIVE","INACTIVE"]
    currencies N string Filter result by list currencies (comma separated value) using valid currency, example: IDR,USD
    accountChannels N string Filter result by list of accountChannels (comma separated value), example: BCA
    requestedBy N string Filter result by user email who create the beneficiary account, example: admin@dashboard.com
    createdTimeStart N string Filter result where beneficiary account created time >= createdTimeStart, example: 2024-08-08
    createdTimeEnd N string Filter result where beneficiary account created time <= createdTimeEnd, example: 2024-08-17
    updatedTimeStart N string Filter result where beneficiary account created time >= updatedTimeStart, example: 2024-08-08
    updatedTimeEnd N string Filter result where beneficiary account created time <= updatedTimeEnd, example: 2024-08-17
    search N string Search beneficiary account by account name or account detail (case insensitive), example: lorem
